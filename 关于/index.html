Hugo, the world's fastest framework for building websites

2019-02-28
阅读时长: 1 分钟

Hugo 采用 Go 语言开发，是一款基于Apache License 2.0. 许可证 开源的静态网站生成器。它支持 TOML、YAML、JSON 格式的数据文件，以及 Markdown、HTML 格式的内容文件，并可通过短代码拓展丰富内容。其核心特色还包括：分类体系、多语言模式、图片处理、自定义输出格式、HTML/CSS/JS 代码压缩，以及 Sass、SCSS 开发流程适配。 Hugo 依托多款开源项目进行开发，包括：

  • https://github.com/yuin/goldmark
  • https://github.com/alecthomas/chroma
  • https://github.com/muesli/smartcrop
  • https://github.com/spf13/cobra
  • https://github.com/spf13/viper

Hugo 非常适合搭建博客、企业官网、创意作品集、在线杂志、单页应用，甚至拥有数千页面的大型网站。

Hugo 专为希望手动编写网站代码，又无需费心配置复杂运行环境、依赖项与数据库的用户而设计。

使用 Hugo 搭建的网站运行速度极快、安全性高，可部署在任意平台，包括亚马逊云、GitHub Pages、Heroku、Netlify 及各类主流托管服务商。

前往 GitHub 了解更多并参与贡献。
